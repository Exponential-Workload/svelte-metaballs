svelte-metaballs

Welcome

Welcome to Svelte-Metaballs!

Svelte-Metaballs is a small(-ish) Svelte-based (duh) Metaballs library that uses WebGL to render things.

Installation

To install Svelte-Metaballs, simply run:

pnpm

pnpm i svelte-metaballs

Yarn

yarn add svelte-metaballs

npm

npm i svelte-metaballs

Usage

Using svelte-metaballs is very simple. Simply import the component & it will automatically size itself to it’s parent.

Notice: it will set the parent’s position to relative. If this is a problem, pass the danger_noForceRelativePosition property. This will cause a mismatch between the canvas’ actual size and the size it’s being rendered at, unless the size of the nearest position: relative ancestor is the same as the direct parent.

The component takes a few arguments; mainly numMetaballs to describe the amount of metaballs to use. I’ll document the rest later

Examples

Fullscreen

Fullscreen usage is as simple as: 
<script lang="ts">
  import Metaballs from 'svelte-metaballs';
</script>

<div class="container">
  <div class="meatballs">
    <Metaballs />
  </div>
</div>

<style lang="scss">
  .container {
    position: fixed;
    top: 0;
    left: 0;
    width: 100vw;
    height: 100vh;
    background: #1a1a1a;
    .meatballs {
      width: 100vw;
      height: 100vh;
      position: relative;
    }
  }
</style>

Sized

If you want to size the metaballs yourself, you can do so: 
<script lang="ts">
  import Metaballs from '$lib/Metaballs.svelte';
</script>

<div class="container">
  <div class="meatballs">
    <Metaballs numMetaballs="{10}" />
    <!-- We set the metaball count to 10 as otherwise we just end up with a gradient at this tiny size -->
  </div>
</div>

<style lang="scss">
  .container {
    position: fixed;
    top: 0;
    left: 0;
    width: 100vw;
    height: 100vh;
    display: flex;
    justify-content: center;
    align-items: center;
    background: #1a1a1a;
    .meatballs {
      width: 400px;
      height: 400px;
      position: relative;
      background: #ffffff11; /* Added for illustrative purposes to show the bounding box of the parent element */
    }
  }
</style>
